# Raycast 音声入力拡張機能 - 要件定義

## 概要

Raycast から呼び出せるローカル音声入力ツール。マイクで話した内容をリアルタイムに文字起こしし、カーソル位置に挿入する。Whisper (tiny/base) をローカル実行し、完全オフラインで動作する。

## ゴール

- **ワンアクションで起動**: Raycast ホットキーで即座に録音開始
- **リアルタイムプレビュー**: 話しながら文字が表示される
- **どこでも入力**: 任意のアプリのカーソル位置にテキスト挿入
- **完全ローカル**: 音声データが外部に送信されない

## ユーザーストーリー

1. ユーザーが Raycast で拡張を呼び出す
2. 即座にマイク録音が開始される
3. 話した内容がリアルタイムでプレビュー表示される
4. Enter / ホットキーで確定 → 直前にフォーカスしていたアプリのカーソル位置にテキスト挿入
5. Esc でキャンセル

## 技術アーキテクチャ

### スタック

| レイヤー | 技術 | 役割 |
|---|---|---|
| UI | Raycast API (React + TypeScript) | 録音制御、プレビュー表示 |
| 音声キャプチャ | ffmpeg (`avfoundation`) | macOS マイクから PCM 16kHz mono 取得 |
| 音声認識 | whisper.cpp (バッチ) | ローカル文字起こし |
| テキスト挿入 | `Clipboard.paste()` | カーソル位置への挿入 |

### リアルタイム処理フロー

```
[マイク] → ffmpeg (avfoundation, 16kHz mono, pipe:1)
    → Node.js バッファ蓄積
    → 3秒ごとに flush → WAV ファイル書き出し
    → whisper-cli で文字起こし
    → React state 更新 → UI プレビュー
    → Enter で確定 → Clipboard.paste()
```

### バイナリ管理

- whisper-cli: ユーザーがビルド or 初回ダウンロード → `environment.supportPath` に配置
- Whisper モデル: 初回起動時に HuggingFace から自動ダウンロード
- ffmpeg: Homebrew 前提 (`brew install ffmpeg`)

## 機能要件

### MVP (v0.1)

- [x] Raycast コマンドで録音開始（自動開始）
- [x] ffmpeg でマイク音声をキャプチャ (16kHz, mono, PCM)
- [x] whisper-cli でチャンクごとに文字起こし（3秒間隔）
- [x] `<Detail>` ビューでリアルタイムプレビュー表示
- [x] Enter で確定 → `Clipboard.paste()` でカーソル位置に挿入
- [x] Esc でキャンセル（録音停止）
- [x] 日本語対応 (`--language ja`)
- [x] セットアップコマンド（依存関係チェック + モデルダウンロード）
- [ ] 初回セットアップ（whisper-cli バイナリの自動ダウンロード）

### v0.2 (改善)

- [ ] 無音検出 (VAD) で自動チャンク分割
- [ ] 認識結果の編集機能（挿入前にテキスト修正）
- [ ] モデルサイズの設定切り替え (tiny / base / small)
- [ ] 録音レベルメーターの表示
- [ ] ショートカットキーカスタマイズ

### v0.3 (拡張)

- [ ] 句読点の自動挿入
- [ ] 連続入力モード（確定後も録音継続）
- [ ] 履歴機能（過去の音声入力を再利用）
- [ ] テンプレート機能（定型文の音声トリガー）

## 非機能要件

- **レイテンシ**: 発話から文字表示まで ~3秒（チャンク間隔） + ~0.5秒（認識処理）
- **プライバシー**: 音声データはローカルのみ。ネットワーク通信なし（モデルDL時を除く）
- **ストレージ**: モデル (tiny: ~75MB, base: ~142MB) + whisper-cli (~10MB)
- **対応 OS**: macOS のみ（Apple Silicon 最適化）
- **依存**: ffmpeg (Homebrew)

## 既存拡張との差別化

| | Whisper Dictation (既存) | 本拡張 |
|---|---|---|
| 認識方式 | バッチ（録音→変換） | チャンクストリーミング |
| プレビュー | なし（結果のみ） | 話しながら文字が出る |
| 音声処理 | sox | ffmpeg |
| セットアップ | brew install 手動 | セットアップコマンド付き |
